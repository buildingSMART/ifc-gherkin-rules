   005 | Feature: GRF001 - Identical coordinate operations
   006 | The rule verifies that if multiple instances of IfcGeometricRepresentationContext are provided to the IfcProject,
   007 | all contexts must share an identical instance of IfcCoordinateOperation, each referring to the same instance of IfcCoordinateReferenceSystem.
   008 | Additionally, the schema allows the attribute for coordinate operations within the context to be empty.
   009 | Currently, for GRF001, this is only permitted if (1) there is a single context or (2) the attribute is empty across all contexts within the file.
   010 |
