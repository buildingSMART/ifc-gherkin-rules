   005 | Feature: PSE002 - Custom properties and property sets validation
   006 | The rule verifies that property set names do not start with any upper- and lowercase variation of 'pset' in order to prevent confusion with standardized property sets in the IFC specification.
   007 | The property sets defined in the IFC specification all start with 'Pset\_'
   008 | 
   009 |     Scenario: Raise a warning for unstandardized property set names
