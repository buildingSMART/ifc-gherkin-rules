   006 | Feature: BLT002 - Correct use of partitioning type attributes for windows
   007 | 
   008 |   The rule verifies that attribute UserDefinedPartitioningType is provided only when
   009 |   the value of the attribute PartitioningType is set to USERDEFINED.
   010 |   Additionally, when an IfcWindow is assigned an IfcWindowType, the type should
   011 |   dictate the PartitioningType, and the instance should not override it.
