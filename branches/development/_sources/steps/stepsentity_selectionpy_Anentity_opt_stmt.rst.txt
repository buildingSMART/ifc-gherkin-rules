An \ .\ ``entity_opt_stmt``\ .\ 
================================

   | 
   | Examples:
   | Given an .IfcAlignment.
   | Given an .IfcRoot. with subtypes
   | Given an .entity instance.
   | 
   | The last example returns everything in the model.
