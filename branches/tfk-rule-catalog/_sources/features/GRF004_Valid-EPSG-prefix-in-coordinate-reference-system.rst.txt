   005 | Feature: GRF004 - Valid EPSG prefix in coordinate reference system
   006 | The rule verifies that if the name of a CRS starts with "EPSG:", it must refer to a valid code from the official EPSG geodetic parameter dataset.
   007 | EPSG code validation is performed using the pyproj library, which includes a local copy of the official EPSG dataset (https://epsg.org) maintained by IOGP.
   008 | For reference: https://pyproj4.github.io/pyproj/stable/api/database.html
   009 |
